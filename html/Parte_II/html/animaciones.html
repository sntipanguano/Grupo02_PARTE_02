Acerca de Animaciones JS DOM

Puede usar JavaScript para crear una animación compleja que tenga, entre otros, los siguientes elementos:

  • Fuegos artificiales
  • Efecto de desvanecimiento
  • Roll-in o Roll-out
  • Entrada de página o Salida de página
  • Movimientos de Objetos

JavaScript se puede utilizar para mover una serie de elementos DOM o cualquier otro elemento HTML, por la página de acuerdo con algún tipo de patrón determinado por una función o ecuación lógica.

JavaScript incluye las tres funciones mencionadas a continuación, que se usan comúnmente en los programas de animación.

  • setTimeout (función, duración):
    • Esta función llama a la función después de la duración en milisegundos a partir de ahora.
  • setInterval (función, duración):
    • Esta función llama a la función después de cada milisegundos de duración .
  • clearTimeout(setTimeout_variable):
    • esta función llama a borrar cualquier temporizador establecido por las funciones setTimeout().

JavaScript también puede establecer una serie de atributos de un objeto DOM, incluida su posición en la pantalla. Puede configurar el atributo superior e izquierdo de un objeto para colocarlo en cualquier lugar de la pantalla. Aquí está su sintaxis.

// Establecer la distancia desde el lado izquierdo de la pantalla.
object.style.left = distancia en pixeles o puntos.
//Establecer la distancia desde el lado superior de la pantalla.
object.style.top = distancia en pixeles o puntos.

Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento.Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua.

Código

train.onclick = function() {
let start = Date.now();
let timer = setInterval(function() {
let timePassed = Date.now() - start;
train.style.left = timePassed / 5 + 'px';
if (timePassed > 2000) clearInterval(timer);
}, 20);
}
